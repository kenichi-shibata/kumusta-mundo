# Kumusta Mundo

Hello World in Tagalog is Kumusta Mundo.

![](app/static/img/cover.png)

## Prerequisites

* python3
* pip3
* pipenv

## Usage

```
cd kumusta-mundo/
pipenv shell
pipenv install
export FLASK_APP=hello-world.py

flask run
```
